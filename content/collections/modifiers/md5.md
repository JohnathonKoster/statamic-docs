Creates an md5 hash of a variable.

::tabs

::tab antlers
```antlers
{{ "hello" | md5 }}
```
::tab blade
```blade
{{ Statamic::modify('hello')->md5() }}

-- or --

{{ md5('hello') }}
```

